Retrocompiler Using:

The objective of the program is to retrocompile bytecodes in java

Option -Target:
Rewrite the bytecode into the version of Java you select.
You must select a java version between 5 and 13 which you want to recompile your file.
--force: is a sub-option to "force" to rewrite features which are not allowed by the version you chose

Option -Infos:
List the features contained in the byte code of the files in entry.
You can chose specific features to list with the option -Features

Option -Features:
- Concatenation: JAVA
- Try With Ressources: JAVA
- Lambda : JAVA
- NestMates: JAVA
- Record :JAVA

Input Path:
This is the path that the program will parse to rewrite(target) or to detect features(infos)
You can enter the path of a directory, a jar or a file .class
